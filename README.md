# rustway
Step by Step learning path to Rust

### Project creation
Creating the new project using the cargo

```shell
cargo new <project_name>
```
These project_name/package name must follow these rules
- Can contains letters,numbers and underscores
- Cannot start with a digit
- Cannot contain hypens in crate names
